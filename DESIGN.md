# poke_fantasy — 게임 기획 · 상세 사양 (정본)

이 문서는 현재 게임의 기획 내용과 세부 동작을 한곳에 정리한 **정본(source of truth)**이다.
시간순 패치 이력은 `BALANCE.md`를 참고한다.

> 규칙 원칙: 플레이어에게만 적용되는 규칙은 상대 AI에게도 항상 대칭으로 적용한다.

---

## 1. 개요

- 장르: 그리드 전술 RPG(무기 종류로 전직이 결정되는 weapon-kind grid tactics).
- 맵: 10×10 타일. 팀당 최대 4명. A팀=플레이어(화면 아래), B팀=AI(화면 위)로 **상하 대결**.
- 화면 흐름: **홈(팀 선택)** → 전투 시작(바로 전투) / 인벤토리(별도 세팅) → **전투** → **결과**.

## 2. 화면 · UI

- **홈(팀 선택)**: A/B팀 캐릭터 선택. "⚔ 전투 시작"으로 바로 전투. "🎒 인벤토리"로 세팅 화면.
- **인벤토리**: 캐릭터별 능력치(표시) · 무기/보조장비 장착 · **기술 로드아웃(최대 4개)** 설정. 로스터 인스턴스를 직접 편집하고, 전투 시작 시 고유 id로 복제해 사용(양 팀 중복 선택 대비).
- **전투 화면(세로 구성: 상단 바 — 맵 — 하단 바로 화면 꽉 채움)**
  - **상단 바**: `🕑 시간대 · 날씨` + **행동 순서 바**(살아있는 전원을 스피드순 정렬, 현재 차례 ▶·금색 강조, 암흑(시야 밖) 상대는 `???`).
  - **맵(카메라 뷰포트)**: 타일 86px 고정, 보드가 화면보다 크며 **카메라가 현재 유닛을 따라 스크롤**(스크롤바 숨김, 숨은 상대 차례엔 이동 안 함). 스타크래프트식 시점.
  - **하단 행동 일자바(약 2배 두께)**: 좌측 상단 = 현재 캐릭터 정보 + 전투 로그 한 줄, **우측 상단 = 버프/디버프 칩(+남은 턴)**(버프 녹색·디버프 적색). 아래에 **기술 2×2 그리드** + 우측(무기교체 / 취소·확인).
  - **아군 정보 카드**: 맵에서 아군(현재 차례가 아닌 아군 포함)을 누르면 좌측 상단에 정보 카드 표시(원화·이름·Lv·무기/방패·HP·능력치·이동/시야·버프/디버프). 닫기 버튼 또는 행동 시 닫힘.
- **표현 디테일**
  - 캐릭터는 직업·성별별 PNG 원화로 표시(아군 후면 back, 적 정면 front). 스프라이트 못 찾으면 아무것도 안 그림.
  - 체력바: 아군은 잔량별 초록/노랑/빨강, **적군(B팀)은 항상 빨간색**.
  - 타일 아이콘: 숲 🌲 / 언덕 ⛰️ / 물 🌊 / 바위 🪨 / 불 🔥(평지는 색만).
  - 안개(암흑, 스타크래프트식): **한 번도 못 본 타일**은 균일한 어두운 회색(지형 종류 알 수 없음). **한 번 밝혔던(탐사) 타일**은 시야를 벗어나도 **지형 색을 어둡게 기억**해서 표시(현재 유닛 위치는 안 보임). **현재 보이는 타일**은 지형 전체 색. 날씨·시간대 틴트는 스크롤 안 되는 무대에 올려 균일하게 덮음.
  - 공격 모션: 시전자가 상대 방향으로 돌진(아군 위·적 아래), 피격 대상은 흔들림 + 붉은 섬광.

## 3. 거리 체계 (중요)

- **시야 · 이동 · 사거리 · 기술 범위/반경 = 마름모(맨해튼 거리)**. 대각선 1칸 = 거리 2.
- 예외: **보호(가드) 반경**은 마름모 1칸(직교 인접 4칸).
- 스킬 명중 판정의 정렬/근접성 등 AI 내부 계산도 맨해튼 기준.

## 4. 캐릭터 · 스탯

- 스탯: `hp` / `attack`(근력) / `magicAttack`(지력) / `defense`(방어) / `speed`(스피드).
- 파생: 레벨(시작 1, 최대 100), `rawMove`(기본 이동력), `sight`(기본 시야 = **전원 5로 통일**).
- `weaponMastery`(무기별 전직 티어 0~6), `skillLoadout`(전투에 들고 갈 스킬 최대 4).
- 로스터(10명, 전원 sight 5): 환도무사·대검전사(검), 철퇴호위·대곤파괴자(둔기), 각궁사수·장궁저격수(활), 화염술사·대지술사(지팡이), 사제·현자(마법서). 각자 rawMove·스탯 상이.

## 5. 이동

- 실제 이동 예산 = `effectiveMove` = rawMove + 보정, 소수점은 내림(floor)해서 칸 수로 사용.
- **마름모(직교 4방향 BFS)**: 대각선은 직교 2칸을 거치므로 비용 2.
- 지형: **물·언덕은 진입 가능하지만 넘어서 이동 불가**(경로가 그 칸에서 끝남). **바위는 진입 불가**(장애물). 점유 타일 진입 불가.
- 보정: **물 위 -1**(급류 시 +1로 상쇄), 다리부상(legHit) magnitude, 날씨(비=중장 -0.5, 눈=경장 -0.5).

## 6. 시야 · 전장의 안개

- `effectiveSight` = 기본 시야 + 보정, 최소 1.
  - 천리안(farSight) +1, **언덕 위 +1**, **밤 -2**, **비/눈 -1**.
- 전장의 안개(스타크래프트식 3단계): 현재 시야=지형 전체 색 / 탐사 완료(과거에 봤음)=지형 색을 어둡게 기억 / 미탐사=균일한 암흑(지형 종류 알 수 없음). 탐사 완료 구역엔 현재 유닛 위치는 표시되지 않음.
- **숲 은폐**: 숲 안 유닛은 (투시 없으면) 맨해튼 1칸 밖에서는 안 보임.
- **불 조명**: 불타는 타일 포함 주변 1칸의 캐릭터/타일은 시야와 무관하게 확인 가능(양 진영).

## 7. 전투 흐름

- 턴 순서: `effectiveSpeed`(스피드×각성보정 + 무기 baseSpeed) 내림차순, 동률은 rng.
- 데미지: `calculateDamage`(속성 상성·주스탯/무기·방어·방패·급소 등 반영). 상세는 `damage.ts`.
- **속성 상성**(불<물<나무<강철<땅<불): 약점 1.3배, 강점 0.7배, 무속성은 영향 없음.
- **기술 로드아웃**: 실제 전투에선 로드아웃(∩ 사용 가능)만 사용. 하나도 없으면 **기본 공격 주먹**(공통/물리/위력 50%/명중 100/사거리 1/무제한, 선택 목록엔 미노출)으로 대체.
- **즉시 발동 UX**: 타겟 필요 없는 기술(자신/자신범위/아군범위)은 버튼만 눌러도 발동. 대상 지정형은 **타겟 클릭 즉시 발동**(확인 불필요). 대기 중 이동이 있으면 이동 후 발동. 확인 버튼은 이동/무기교체/대기(턴 종료)용.
- **무기 교체**: 하단바에서 순환 선택 후 확인. 숙련 티어<3이면 교체가 그 턴을 소모(엔진 규칙).
- 처치·경험치: 막타를 넣은 캐릭터가 경험치 획득, 레벨업 시 전직 포인트 지급.

## 8. 무기 · 전직

- 무기 종류(kind): 검(sword)·둔기(blunt)·창(spear)·활(bow)·석궁(crossbow)·단검(dagger)·투척(thrown)·지팡이(staff)·마법서(tome)·방패(shield). ※ 창/석궁/단검/투척은 타입 예약 중심, 미구현 부분 존재.
  - **근접**: 검·둔기·창·단검. **원거리·마법**: 활·석궁·투척·지팡이·마법서.
- 무기 템플릿(발췌): 환도/대검(검 r1), 철퇴/대곤(둔기 r1), 각궁/장궁(활 r2), 법장/스태프(지팡이 r2), 주술서/마법서(마법서 r1), 원형/타워 방패(방어 +15/+25).
- 전직: 5레벨 단위, 최대 티어 6. `getUsableSkillIds`가 티어별 해금 스킬 반환.
  - 티어1 보너스: 검 위력+10%, 둔기 명중+5, 활 명중+10, 지팡이/마법서 위력+10%.
  - 티어5 패시브(설계 placeholder): 검=저체력 위력↑, 둔기=방어↑, 활=시야+1, 지팡이=속성강점 페널티 무시, 마법서=사용횟수+1.

## 9. 스킬 (요약)

- **공통**: 주먹(폴백), 강타(물리 100%), 주술(마법 100%, 지팡이 속성), 보호(주변 1칸 아군 대신 피격 2턴), 도발, 낙석(언덕 전용), 마법부여-불/물/나무/강철/땅, 급류, 등반, 천리안, 투시.
- **검**: 발도(T2 주변 광역), 각성(T4 아군 스피드↑), 고속연타(T6 3연타).
- **둔기**: 다리타격(T2 이동↓), 단결(T4 아군 방어↑), 분쇄(T6 강타).
- **활**: 화공(T2 불 타일 생성), 급소(T4 급소확률), 저격(T6 시야 내 적).
- **지팡이**: 원소탄(T2), 약화(T4 속성 변경), 원소폭풍(T6 광역).
- **마법서**: 치료(T2 회복), 원기회복(T4 사용횟수+1), 재행동(T6).

## 10. 지형 · 타일

1. **평지**: 특수 효과 없음(색: 연두).
2. **숲**: 인접 1칸 밖에서 은폐(내부 유닛 안 보임). **숲 안에서는 근접 무기만 행동 가능**. 진입/통과 가능. 불타면 평지로 변함.
3. **물**: 진입 가능하나 **넘어서 이동 불가**. **물 위 이동력 -1**(급류로 상쇄).
4. **언덕**: **등반 버프 없이도 진입 가능**, 넘어서 이동 불가, **언덕 위 시야 +1**, **올라선 턴엔 행동 불가**(등반 버프 시 예외).
5. **바위**: 장애물(진입 불가). **원거리·마법 공격이 바위를 넘어 타격 불가**(직선 경로 차단).
6. **불(타일 상태)**: 해당 타일 캐릭터 매 턴 최대체력 1/4 피해. 주변 1칸 조명(시야 무관 확인). 지속 후 소멸.

## 11. 날씨 · 시간대

- **날씨**(전투 시작 시 랜덤): 맑음 / 비 / 눈 / 폭염.
  - 비: 중장(검·둔기·창·방패) 이동 -0.5, 시야 -1.
  - 눈: 경장(원거리·마법) 이동 -0.5, 시야 -1.
  - 폭염: 턴 시작 시 최대체력 1/16 피해(정신력=마법공격력 기반, 최대 70% 확률로 저항).
- **시간대**(랜덤): 아침 / 낮 / 저녁 / 밤. **밤 시야 -2**. 보드에 옅은 색 틴트.

## 12. 상태이상 (StatusEffectType)

- 버프: guarding(보호), elementEnchant(마법부여), riverSurge(급류), climbing(등반), farSight(천리안), forestVision(투시), swordAwaken(각성), bluntUnity(단결), bowCrit(급소), focused(집중).
- 디버프: taunted(도발), legHit(다리부상).
- 각 상태는 남은 턴 수를 가지며 턴 시작에 감소. 하단바 우측 상단 칩으로 표시.
