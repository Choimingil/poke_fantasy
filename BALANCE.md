TRPG 전투의 능력치·방어구·이동력 등 세부 수치와 변경 이력을 관리하는 문서입니다.
수치 조정(밸런스) 관련 변경이 있을 때마다 아래 **패치 로그**에 계속 추가합니다.

관련 상수 위치: `src/game/trpg/engine.ts` (`ARMOR_STATS`, `MOVE_CAP`, `MOVE_POINTS_PER_TILE`, `baseMoveFor`), `src/game/trpg/map.ts`(지형).

## 능력치 개요

| 능력치 | 설명 |
| --- | --- |
| 체력(HP) | 0이 되면 묘지로 이동(사용 불가) |
| 공격력 | 물리(근거리/원거리) 데미지 기준 |
| 마력 | 마법 데미지 기준 |
| 방어력 | 받는 데미지 감소 (방어구 보너스 포함) |
| 스피드 | 행동 순서(높을수록 먼저) |
| 이동력 | 한 턴 이동 가능 칸수(아래 공식) |
| 시야 | 마름모 반경만큼의 타일/적을 인지, 밖은 암흑 |
| 사거리 | 무기별 공격 범위(검1·활2·지팡이1, 맨해튼 거리) |

## 방어구

| 종류 | 방어력 | 이동력 제한 | 무게 |
| --- | --- | --- | --- |
| 천(cloth) | +1 | 0 | 경장 |
| 가죽(leather) | +2 | −0.5 | 경장 |
| 중갑(mail) | +3 | −1 | 중장 |
| 판금(plate) | +4 | −1.5 | 중장 |

- 기본 착용: **전사=판금, 격수=가죽, 법사=천**.
- 전투 중 무기와 함께 **교체 가능**(교체 시 그 턴 공격 불가 = 행동 종료).
- 경장(천/가죽) = 눈 피해 대상, 중장(중갑/판금) = 비 이동 페널티/눈 면역 대상.

---

## 이동력 공식

```
rawMove   = 기본이동력 + floor(전직 이동강화 포인트 / 2)     // 2점당 +1칸
이동페널티 = 방어구 제한 + (물 위 1) + (비 & 중갑/판금 1)
E(유효)   = min(3, rawMove − 이동페널티)                     // 소수 허용
이동 칸수 = floor(E)
```

- **기본 이동력(방어구 적용 전): 전사 4, 격수 3, 법사 2** → 기본 방어구 착용 시 유효 이동력은 모두 **2칸**.
- **상한 3**: 아무리 투자해도 유효 이동력은 최대 3칸.
- **초과분 완충**: `rawMove`가 3을 넘으면 그 초과분이 이동 페널티를 **먼저 흡수**(속도 인플레 없이 악천후/중장 대응).
- **이동 불능**: `floor(E) < 1`이면 그 턴에는 **이동(등반 포함)만 불가**, 공격·무기/방어구 교체 등 다른 행동은 가능. 페널티가 지속되면 자연히 여러 턴 유지.
- **전직 이동 강화 비율**: 강화 1포인트 = 실제 +0.5칸(2점당 1칸). +1 강화가 곧 +1칸이 되지 않도록 완만화.

### 예시 (맑음 기준)

| 유닛/방어구 | rawMove | 페널티 | E | 이동 칸 |
| --- | --- | --- | --- | --- |
| 전사·판금 | 4 | 1.5 | 2.5 | 2 |
| 전사·중갑 | 4 | 1 | 3 | 3 |
| 전사·천 | 4 | 0 | 3(상한) | 3 |
| 격수·가죽 | 3 | 0.5 | 2.5 | 2 |
| 법사·천 | 2 | 0 | 2 | 2 |
| 법사·판금 | 2 | 1.5 | 0.5 | 0(이동 불능) |
| 전사·판금 + 비 | 4 | 2.5 | 1.5 | 1 |
| 전사·판금 + 이동강화 2점 | 5 | 1.5 | 3 | 3 |

## 시야 · 날씨 · 시간대 · 지형 (요약)

- **시야**: 기본 5(마름모). 밤=2로 제한, 비/눈 −1(중첩 시 `min`). 시야 밖은 암흑 — 적 위치·정체·행동 비노출.
- **시간대**: 낮(무효과) / 밤(2칸 밖 암흑).
- **날씨**: 맑음(무효과) / 비(시야 −1, 중갑·판금 이동 −1) / 눈(시야 −1, 천·가죽 매 라운드 HP 1/16 감소).
- **지형**: 평지 / 나무(그 위: 활 공격 방어 0.5배 + 활 시야 차단) / 물(그 칸: 이동 −1) / 바위(통과 불가 + 시야 차단) / 언덕·산(진입 비용 2 = 등반, 산 위: 활 공격력 +30%).
- **맵**: 10×10 무작위 생성, 시작 칸 주변 평지 보정, 두 진영 간 걸어갈 수 있는 통로 보장.

## 패치 로그

### 2026-07-14
- **방어구 재정의**: 방어력 천 +1 / 가죽 +2 / 중갑 +3 / 판금 +4, 이동력 제한 0 / −0.5 / −1 / −1.5.
- **이동력 공식 재조정**: 기본 이동력 전사 4·격수 3·법사 2(기본 방어구 시 유효 2칸), 유효 이동력 `E = min(3, rawMove − 페널티)`에 **소수 페널티** 반영 후 `floor(E)`로 칸수 계산.
- 눈 HP 감소 대상을 **천·가죽**으로 명시.

### 이전 (같은 작업 반영분)
- 이동력 **상한 3** 도입, 초과분은 페널티 **완충**으로만 사용.
- 전직 이동 강화 **비율 2점 = +1칸**(`MOVE_POINTS_PER_TILE`) — +1 강화가 과도해지지 않도록.
- 유효 이동력 1 미만 시 **바닥값(1) 보장 제거** → 그 턴 이동만 불가(행동 가능).
- 비 이동 −1 대상을 **중갑·판금**으로 지정.
