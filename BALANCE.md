# 밸런스 · 패치 로그

> 원칙: 플레이어에게만 적용되는 규칙은 상대 AI에게도 자동으로 동일하게 적용한다.
> (시야/안개, 날씨 이동 페널티, 폭염 피해 등은 양 진영 대칭)

이 문서는 그리드 전술 RPG(weapon-kind grid tactics) 엔진 기준의 밸런스·표현 계층 변경 이력을 기록한다.

---

## 전투 표시 — 피격 대상 위 데미지 숫자·빗나감

- 공격 시 피격 대상 위에 **입힌 데미지 숫자**를 띄운다(급소는 크게·붉은색+`!`). **회피/명중 실패 시 "빗나감"**, 치료 시 **+회복량**(초록)도 표시.
- 엔진이 턴마다 `GridBattle.lastTurnEvents`(대상별 데미지/빗나감/회복)를 누적하고, UI가 대상별로 합산해 유닛 위에 잠깐 떠오르는 플로팅 텍스트로 렌더(다단 히트·범위 피해는 합산).

## 테스트 로스터 개편 — 동양풍 인명·무기별 1인·레벨30 6차전직

- 로스터 10명을 **직업명 → 동양풍 인명**으로 변경(강도현·백건우·서지훈·윤재하·한도영·류시아·오세준·임하늘·문서준·정유안).
- 각 캐릭터가 **서로 다른 무기 1종**을 장착(검·둔기·창·활·석궁·단검·투척·불 지팡이·물 지팡이·마법서 — 9개 무기 종류 전부 커버, 지팡이는 불/물 2종).
- 전원 **레벨 30**, 장착 무기를 **6차 전직**(티어6, 초/중/고급 전용기술 전부 사용 가능). 기본 로드아웃도 해당 무기 전용기술 3종 + 기본 공격.
- 능력치·장비를 레벨/무기에 맞게 조정: 5개 능력치 합 112(레벨30 분배 포인트 87), 무기 주스탯 위주 배분, 무기·방어구·방패 Lv.30(적재량 내), `createCharacter`에 `weaponMastery`·`skillLoadout` 옵션 추가.

## 전투 시스템 전면 개편 — 무기별 전용기술 27종·전직 패시브 9종·공통 규칙

- **전용 기술 전면 재정의**: 9개 무기(검·둔기·창·활·석궁·마법서·지팡이·단검·투척) 모두 **초(T2)/중(T4)/고급(T6) 3개씩** 구현. 기존 검·둔기·활·지팡이·마법서 스킬(발도·각성·고속연타·단결·분쇄·화공·급소·원기회복 등)은 새 스킬로 교체, 창·석궁·단검·투척은 신규.
  - 검: 반월참(부채꼴)·일섬(넉백)·섬광참(돌진). 둔기: 다리타격(이동-1)·밀쳐내기(넉백/기절)·광역보호(경호 강화). 창: 꿰뚫기(관통 2타)·봉쇄(이동불가)·돌진(방패 무력화). 활: 천궁(언덕 사거리+1)·도약사격(추가이동)·저격(5칸/차단). 석궁: 철갑사격(방어 20% 무시)·관통사격(직선 2타)·치명사격(최대체력 50% 고정피해). 마법서: 치료·정화(디버프 제거)·재행동. 지팡이: 원소탄·약화·원소폭풍(속성별 이름 자동변환, 광역 중심150/주변100). 단검: 기습(추가이동)·은신(투명)·축지(아군 인접 순간이동). 투척: 맹독(지속피해, 출혈 중복)·분신(0.3배 추가타)·쇄상(디버프 대상 ×1.3).
- **공통 기술 개정**: 보호·급류·등반·천리안·투시는 지속 중 재사용 불가, 마법부여 주스탯이 **max(근,지)+0.5×min(근,지)**로 변경, **빠른교체** 신규(3턴 무기 교체 무료·턴당 1회·전용기술 잠금).
- **전직 티어 의미 정리**: 1차 숙련도, 2차 초급, 3차 무기교체 무료, 4차 중급, 5차 패시브, 6차 고급. **티어5 패시브 9종 신설**: 질주(검)·경호(둔기)·전열유지(창)·매의눈(활)·정조준(석궁)·명상(마법서)·증폭(지팡이)·적응력(단검)·협공(투척). (기존 저체력/방어/시야/속성방벽/사용횟수 패시브는 대체)
- **신규 상태이상**: 광역보호·봉쇄·은신·분신·맹독·빠른교체. 제거: 각성·단결·급소(스킬 상태).
- **반응·위치이동 시스템**: 경호/협공/분신 반응 공격(라운드당 1회, 추가공격의 추가공격 재발동 차단), 도약사격·기습 공격 후 추가이동(플레이어 수동 선택), 축지 순간이동 후 추가행동, 은신 투명(시야 제외·이동 차단·공격/피격 시 해제).
- **공통 전투 규칙**: 고정피해는 방어/속성/급소/위력증감 무시, 0% 보조기술 위력증가 미적용, 추가공격 재귀 차단, 동명 강화/약화 미중첩(지속시간 갱신), 1회 기술은 보존/회복 대상 아님. **보스 개념 미도입**(보스 감소 조항 생략).
- UI: 상태칩 신규 상태 라벨·설명 추가, 지팡이 원소 스킬 이름 자동변환 표시, 축지 목적지·후속 이동 칸 하이라이트.
- 참고: Playwright 미설치 → UI(후속 이동·축지 하이라이트, 상태칩)는 typecheck·vitest(113개)·build로 검증. 시각 확인은 사용자 확인 필요.

## 양손 파지 무기공격력 1.3배

- 단검·투척무기를 제외한 모든 무기는 방패(보조 무기) 없이 두 손으로 들 수 있고, **양손 파지 시 무기공격력 ×1.3**(`effectiveWeaponPower`). 데미지 공식의 무기공격력에 반영(플레이어·AI 대칭).
  - 양손 전용 무기(대검·창·활·석궁·지팡이)는 항상 적용, 한손 무기(환도·철퇴·주술서·마법서)는 방패를 빼면 적용, 단검·투척은 미적용.
- 인벤토리 창·전투 장비교체 창에 각 무기의 공격력 수치와 "양손 ×1.3(적용)" 배지를 표시.

## 상태 설명 · 조회 팝업 전환 · 무기 속성/부가효과 표시

- **상태 정보 설명**: 하단 바 버프/디버프 칩을 누르면 해당 상태의 뜻을 설명하는 한 줄이 아래에 표시(다시 누르면 닫힘). 14종 상태이상 전부에 설명 문구(`STATUS_DESC`) 부여, 마우스오버 툴팁으로도 노출.
- **조회 팝업 동작**: 캐릭터 정보 카드가 떠 있을 때 **다른 캐릭터를 누르면 그 캐릭터로 자동 전환**, **빈 타일을 누르면 팝업이 닫힘**. 조회 대상은 아군 전체 + 시야 안의 적군.
- **무기 인스턴스 표시 이름**(`weaponInstanceName`): 지팡이는 속성을 붙여("불의 법장"), 마법서·투척무기는 부가효과를 붙여("출혈의 주술서", "기절의 투척무기") 표시. 인벤토리·장비교체 창·정보 카드·하단바에 일괄 적용.
- 참고: 마법서/투척 인스턴스별 부가효과, 지팡이 인스턴스별 속성(+초급/고급·주술 기술이 지팡이 속성 상속), 속성 상성(불<물<나무<강철<땅<불; 약점 ×1.3·강점 ×0.7·무속성 무영향) **로직은 이미 엔진에 구현되어 있었고**, 이번 변경은 이를 UI에 노출·표시하는 표현 계층 작업이다.

## 날씨 이동 페널티를 방어구 기준으로 수정(버그)

- 비/눈 이동 페널티가 **무기 종류**로 판정되던 버그 수정 → **착용 방어구**로 판정. 비=중갑·판금(-0.5), 눈=천·가죽·미착용(-0.5). 이제 판금→가죽으로 갈아입으면 비 페널티가 사라진다.

## 기술 타입 · 장비교체 창 · 이동 불가 안내

- **기술 타입(물리/마법/변화)** 분류(`skillTypeLabel`) 추가 및 표시(인벤토리 목록 + 전투 버튼 배지). 주술·지팡이·마법서 전용=마법, 그 외 위력 있음=물리, 위력 없음=변화. (데미지 주스탯은 물리=근력·마법=지력으로 기존 damageType과 일치)
- **장비교체 창**: 하단 "장비교체"가 순환 선택에서 **모달 창**으로 변경. 보유 무기/방어구를 보여주고(착용 레벨 미달은 잠금) 하나 선택 후 완료 시 교체+턴 종료.
- **이동 불가 안내**: 이동력 1 미만이라 이번 턴 이동할 수 없을 때 하단 바에 "이동력이 1 미만이라 이번 턴에는 이동할 수 없습니다 (2턴에 1칸)" 안내 표시.

## 레벨업 포인트 분배제 · 데미지/이동 공식 · 무기 부가효과 도입

- **스탯 체계 전면 개편**: 기존 `defense`(방어) 스탯을 제거하고 `endurance`(지구력)를 새 5번째 점수-분배 스탯으로 추가. 점수로 찍을 수 있는 5개 능력치(체력/근력/지력/스피드/지구력)는 전원 초기값 5, 레벨업마다 3포인트를 인벤토리 화면에서 직접 분배(`unspentStatPoints`/`spendStatPoint`, 기존 자동 스탯 성장 제거). 최대 레벨 100 상한 적용.
- **방어력이 완전히 파생 스탯화**: 더 이상 기본 스탯이 아니며 방어구+방패 수치만으로 결정. 데미지 공식도 `최종공격력 − 방어력`(뺄셈) 방식으로 교체(기존 나눗셈식 폐기).
- **데미지 공식 교체**: `최종공격력 = (주스탯/6 + 무기공격력) × 숙련도~100% 랜덤값 × 기술위력`. 숙련도 랜덤값 하한은 전직 티어별 placeholder(티어0=50%~티어6=100%). 방패 방어 보너스 수치를 새 공식 규모에 맞춰 15/25 → 3/6으로 재조정.
- **정신력·회피율 신설**: 정신력 `(지력/305)×0.7`(상대 디버프·부가효과 무시 확률, 최대 70%), 회피율 `(스피드/305)/2 + 레벨차/100`(공격 회피 확률, [0,1] 클램프). 회피 판정은 명중 판정 통과 후 방어측이 한 번 더 굴리며, 활의 '집중' 발동 시 생략.
- **이동력 공식 교체**: `(지구력/30)+1`, 최대 5칸. 5 초과분은 이동 감소 페널티를 먼저 상쇄하는 버퍼로만 작용(END가 높을수록 슬로우 저항력↑). 이동력 1 미만이면 2턴에 1칸(홀수 라운드에만).
- **무기 부가효과 신설**(적중 시 30% 확률 발동): 검=출혈(2턴 DoT 1/8), 둔기=기절(2턴간 매 턴 30% 별도 판정), 창=관통(방어력 전부 무시), 활=집중(회피율 무시), 석궁=급소(×1.5), 방패=없음. 마법서·투척무기는 인스턴스마다 위 5개 중 하나를 선택해 지님. 출혈·기절 적용은 정신력 저항 판정을 거침(기존 도발/다리부상/약화 디버프에도 동일 저항 규칙을 소급 적용).
- **단검 이도류 지원**: 주무기가 단검일 때만 보조 슬롯(방패 슬롯 재사용)에 단검을 하나 더 장착 가능. 다른 무기와는 병용 불가, 보너스 효과 없음(순수 슬롯 허용).
- 로스터 10명의 스탯을 새 5스탯 모델(레벨10 기준 5×5+27=52 포인트)로 재설계.

## 버그 수정 — 시야 밖일 때 AI가 멈춰있던 문제

- **증상**: 교전 전이나 적이 시야를 벗어난 상태에서는 `pickAiAction`이 빈 행동(`{}`)을 반환해 AI가 제자리에서 계속 대기했다. 두 팀이 상하로 멀리 떨어져 시작하는 맵 구조상(기본 시야 5 vs 스폰 거리 약 9) 첫 교전까지 양쪽 다 서로를 못 보는 경우가 흔해, 사실상 아무도 움직이지 않는 것처럼 보였다.
- **수정**: `GridBattle`이 각 진영이 상대 유닛을 마지막으로 확인한 위치를 `knownEnemyPositions`에 기억(시야를 벗어나도 유지)하도록 하고, AI는 시야 안에 실제 대상이 없으면 그 기억 중 가장 가까운 위치로, 그마저 없으면 맵 중앙으로 계속 이동하도록 변경. 공격은 여전히 실제로 시야에 들어오거나 도발로 강제된 대상에게만 시도(추측 위치로 공격하지 않음 — 플레이어와 동일한 정보 제약 유지).

## 장비(무기 · 방어구) 시스템 전면 개편

- **적재량 스탯 신설**: 기본 5kg + 근력 5당 1kg. 장착·소지 중인 모든 무기·방어구 무게 합이 적재량을 넘을 수 없다.
- **무기 공격력을 레벨 기반 공식으로 통일**: 기존 무기 종류별 고정 `basePower`를 제거하고, 모든 무기 인스턴스가 10단위 착용 레벨(10~100)을 가지며 공격력 = `착용 레벨 / 2`(단검은 3/4배)로 계산한다.
- **방어구(3번째 장비 슬롯) 신설**: 천/가죽/중갑/판금 4종. 방어력 = 같은 레벨 무기의 기준공격력 × 종류별 배수(0.7/0.9/1.1/1.3). 무게는 0.5/1/3/5kg. 기존 방패(shield)와는 별개의 슬롯이며 방패 로직은 그대로 유지.
- **착용 레벨 제한**: 캐릭터 레벨이 장비 레벨 이상이어야 장착 가능. 인벤토리에서 미달 항목은 버튼 비활성화 + 필요 레벨 표시.
- **인벤토리 슬롯 제한**: 기본 장착 장비 외 최대 2개까지만 추가 소지 가능(적재량 범위 내).
- **전투 중 장비 교체 통합**: 기존 무기교체 버튼이 무기와 방어구를 하나의 순환 목록으로 함께 보여주도록 확장(레벨 미달 항목은 후보에서 제외). 방어구 교체는 항상 턴을 소모.
- **로스터 시작 레벨 1 → 10**: 전원 Lv.10 무기/방어구로 시작. 창/석궁/단검/투척무기 종류의 기본 무기 템플릿도 추가(전용 스킬은 아직 미구현, 공통 스킬만 사용 가능).
- 데이터 무결성 회귀 테스트(`roster.test.ts`) 추가: 전 로스터 캐릭터의 소지 장비 무게가 적재량 이내, 추가 소지 장비 ≤2개, 기본 장착 장비가 착용 레벨 조건을 만족함을 검증.

## 타일 정보 정비(스펙 반영)

- **바위(rock, 신규)**: 장애물(진입 불가). 원거리·마법 공격이 바위를 넘어 타격 불가(직선 경로에 바위가 있으면 차단). 아이콘 🪨, 맵에 중앙·가장자리 배치.
- **숲(forest)**: 인접 1칸 밖에서는 은폐(기존). 추가로 **숲 안에서는 근접 무기만 행동 가능**(원거리·마법 무기는 스킬 사용 불가).
- **물(water)**: 진입은 되지만 **넘어서 이동 불가**(경로 종착). **물 위 이동력 -1**(급류로 상쇄).
- **언덕(hill)**: 이제 **등반 버프 없이도 올라갈 수 있음**(버그 수정). 넘어서 이동 불가(경로 종착), **언덕 위 시야 +1**, **올라선 턴에는 행동 불가**(등반 버프 시 예외).
- **불(fire)**: 해당 타일 캐릭터 체력 1/4 감소(기존), 숲이 불타면 평지로 변경(점화 시 즉시). 추가로 **불 타일 포함 주변 1칸의 캐릭터는 시야와 무관하게 확인 가능**(양 진영).

- **흐름 개편**: 홈에서 캐릭터 선택 후 바로 전투 시작. 능력치·기술·무기 세팅은 별도 "인벤토리" 버튼에서 진행.
  - 인벤토리는 로스터 인스턴스를 직접 편집하고, 전투 시작 시 고유 id로 복제해 사용(양 팀 중복 선택 대비).

## 표현 계층 복원 + 날씨/지형 재이식 (element-grid-tactics 재작성 이후)

`feature/element-grid-tactics` 재작성으로 기술/전직/속성 시스템은 새 아키텍처로 재구현되었으나,
이전 버전에서 반영했던 표현 계층과 날씨 시스템이 사라져 아래를 다시 반영했다.

### 표현 계층

- **캐릭터별 원화 스프라이트 복원**: 전투 유닛이 단일 플레이스홀더(SVG)로 덮이던 것을 되돌리고,
  직업·성별별 PNG 원화(`assets/jobs/{spriteJob}-{gender}-{front|back}.png`)로 표시.
  - 각 로스터 캐릭터에 `spriteJob`/`gender` 매핑 추가.
  - 아군(A팀)은 후면(back), 상대(B팀)는 정면(front) 방향으로 표시.
  - 스프라이트를 찾지 못하면 아무것도 렌더하지 않아 캐릭터가 사각형 이미지로 덮이지 않는다.
- **카메라 추적 복원**: 전투 보드를 고정 픽셀(64px) 셀로 만들고 스크롤 뷰포트에 담아,
  현재 행동 유닛을 화면 중앙으로 부드럽게 추적. 단, 플레이어에게 보이지 않는(안개/숲 은폐) 상대의
  차례에는 카메라를 옮기지 않고 아군 시점을 유지한다.
- **전장의 안개 / 숲 은폐 복원**:
  - 플레이어(A팀) 시야 밖 타일은 안개로 덮는다. 숲 타일은 실루엣이 항상 보이되 내부 유닛은 은폐.
  - 플레이어 시야에 들어오지 않은 상대 유닛 토큰은 표시하지 않는다.
  - 시야 밖 상대는 타겟팅(공격 대상 지정)도 불가.

### 날씨 (맑음 / 비 / 눈 / 폭염)

- 전투 시작 시 무작위로 하나 결정(양 진영 공통).
- **비**: 중장(근접: 검·둔기·창·방패) 계열 `rawMove` −0.5.
- **눈**: 경장(원거리·마법: 활·석궁·단검·투척·지팡이·마법서) 계열 `rawMove` −0.5.
- **폭염**: 턴 시작 시 최대 체력 1/16 피해. 단, 정신력(마법공격력 기반, 최대 70%) 확률로 저항.
  - 저항 확률 = min(0.7, 마법공격력 / 40).
- **맑음**: 페널티 없음.

이동 페널티는 `effectiveMove`에, 폭염 피해는 턴 시작 처리에 통합되어 플레이어·AI 모두에 대칭 적용된다.
